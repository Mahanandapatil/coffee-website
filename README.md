# Coffee Website

A simple and responsive coffee shop website built using HTML, CSS, and JavaScript.

## Features

- Responsive design
- Hero section with call to action
- About, Menu, Testimonials, Gallery, and Contact sections
- Swiper.js slider for testimonials
- Font Awesome icons

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Swiper.js
- Font Awesome
- Google Fonts
